## Deskripsi Project

Toko Bangunan Rasyid merupakan aplikasi desktop berbasis Java Swing dan MySQL yang digunakan untuk membantu proses pengelolaan data material pada toko bangunan.

## Objek / Domain

Aplikasi ini bergerak pada bidang manajemen toko bangunan, khususnya pengelolaan data material.

## Teknologi yang Digunakan

- Java
- Java Swing
- NetBeans IDE
- MySQL Database

## Fitur Aplikasi

- Login pengguna
- Menampilkan data material
- Menambah data material
- Mengubah data material
- Menghapus data material
- Penyimpanan data menggunakan database MySQL

## Cara Menjalankan Aplikasi

1. Import file database `db_tokobangunan_231011401568.sql` ke MySQL.
2. Sesuaikan konfigurasi koneksi database pada file `Koneksi.java`.
3. Buka project menggunakan NetBeans.
4. Jalankan aplikasi.
